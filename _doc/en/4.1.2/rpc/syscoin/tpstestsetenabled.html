--- name: tpstestsetenabled btcversion: 4.1.2 btcgroup: syscoin permalink: en/doc/4.1.2/rpc/syscoin/tpstestsetenabled/ --- tpstestsetenabled [enabled] Set TPS Test to enabled/disabled state. Must have -tpstest configuration set to make this call. Arguments: 1. enabled (boolean, required) TPS Test enabled state. Set to true for enabled and false for disabled. Example: > syscoin-cli tpstestsetenabled true
