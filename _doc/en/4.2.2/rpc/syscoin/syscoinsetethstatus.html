--- name: syscoinsetethstatus btcversion: 4.2.2 btcgroup: syscoin permalink: en/doc/4.2.2/rpc/syscoin/syscoinsetethstatus/ --- syscoinsetethstatus "syncing_status" highest_block Sets ethereum syncing and network status for indication status of network sync. Arguments: 1. syncing_status (string, required) Syncing status either 'syncing' or 'synced' 2. highest_block (numeric, required) What the highest block height on Ethereum is found to be. Usually coupled with syncing_status of 'syncing'. Set to 0 if sync_status is 'synced' Result: { (json object) "status" : "str" (string) Result } Examples: > syscoin-cli syscoinsetethstatus "syncing" 7000000 > syscoin-cli syscoinsetethstatus "synced" 0 > curl --user myusername --data-binary '{"jsonrpc": "1.0", "id": "curltest", "method": "syscoinsetethstatus", "params": ["syncing", 7000000]}' -H 'content-type: text/plain;' http://127.0.0.1:8370/ > curl --user myusername --data-binary '{"jsonrpc": "1.0", "id": "curltest", "method": "syscoinsetethstatus", "params": ["synced", 0]}' -H 'content-type: text/plain;' http://127.0.0.1:8370/
